빠른 거듭제곱

작성일 2020-10-01 수정일 2020-10-06 카테고리 algorithm Disqus:

개요

양의 정수 aaa와 nnn에 대해서, ana^nan을 구하는 상황을 생각해 봅시다.
단순하게 반복문을 nnn번 돌리는 방법이 있습니다.

하지만 nnn이 10억처럼 큰 수라면 직접 돌리기엔 부담스럽죠.
지금부터 O(n)O(n)O(n)의 시간복잡도를 개선해 봅시다.

빠른 거듭제곱

an+m=an∗ama^{n+m} = a^n * a^m an+m=an∗am

지수법칙을 배우셨다면 위 식이 이해되실 겁니다.
우리는 이렇게 ana^nan을 적절히 찢어서 계산할 건데, 여기서 nnn을 2진수로 표현합니다.
이해를 돕기 위해 3133^{13}313을 예시로 들어보면 다음과 같습니다.

313=311012=38∗34∗313^{13} = 3^{1101_2} = 3^8 * 3^4 * 3^1 313=311012​=38∗34∗31

31=332=(31)2=934=(32)2=8138=(34)2=6561\begin{aligned} &3^1 = 3\\ &3^2 = (3^1)^2 = 9\\ &3^4 = (3^2)^2 = 81\\ &3^8 = (3^4)^2 = 6561 \end{aligned} ​31=332=(31)2=934=(32)2=8138=(34)2=6561​

최종적으로 313=6561∗81∗33^{13} = 6561 * 81 * 3313=6561∗81∗3을 얻을 수 있습니다.
ana^nan을 구하려면 ⌊log⁡2n⌋\lfloor \log_2 n \rfloor⌊log2​n⌋번만 제곱 연산을 수행하면 되므로, 최종 시간복잡도는 O(log⁡n)O(\log n)O(logn)이 됩니다.

구현

오버플로우는 적당히 MOD를 통해 처리하면 됩니다.
